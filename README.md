LootRadar, çeşitli dijital oyun platformlarındaki ücretsiz ve indirime giren oyunları anlık olarak takip edip kullanıcıya haber veren bir Android uygulamasıdır. 

*Not: Bu proje şu anda aktif olarak geliştirilme aşamasındadır.*

## 🚀 Hedeflenen Özellikler (Roadmap)
- [ ] Farklı platformlardaki güncel oyun indirimlerini ve kampanyalarını listeleme.
- [ ] Ücretsiz olan oyunlar için bildirim gönderme.
- [ ] Oyunların eski ve güncel fiyatlarını (TL bazında) karşılaştırmalı olarak gösterme.
- [ ] Kullanıcıların istedikleri oyunları "İstek Listesine" ekleyip fiyat düşüşlerinde haberdar olması.

## 🛠️ Kullanılan Teknolojiler ve Mimari
- **Dil:** Kotlin
- **Kullanıcı Arayüzü (UI):** Jetpack Compose
- **Mimari:** MVVM (Model-View-ViewModel)
- **Ağ/Network:** Retrofit 
- **Yerel Veritabanı:** Room Database
- **Bağımlılık Enjeksiyonu (DI):** Dagger Hilt
